## 📲 Expo Go 앱으로 테스트

### 1단계: Expo Go 앱 설치
- **iOS**: App Store에서 "Expo Go" 검색
- **Android**: Play Store에서 "Expo Go" 검색

### 2단계: 서버 실행 (한 번만)
```bash
# 터미널 1: 백엔드 서버
npx tsx server/index.ts

# 터미널 2: Expo 서버
npx expo start
```

### 3단계: 연결 (QR 코드 스캔)

**`start-dev.bat` 실행 후 "VibeTrip Expo 서버" 창에서 QR 코드 찾기**

- **Android**: Expo Go 앱 열기 → "Scan QR code" 버튼 → QR 스캔
- **iOS**: 기본 카메라 앱으로 QR 코드 스캔 → 알림 터치

---

## ⚠️ 참고사항

- IP 주소는 네트워크 환경에 따라 변경될 수 있음
- 컴퓨터와 휴대폰이 **같은 Wi-Fi**에 연결되어 있어야 함
- IP 확인 명령어: `ipconfig` (Windows)
