# 👤 사용자 입력 요소 정의 (USER_INPUT.md)

## 개요

초개인화 여행일정을 생성하기 위해 수집해야 하는 사용자 정보를 정의합니다.

---

## 1단계: 사용자 프로필 (Profile)

### 1.1 기본 정보

| 필드 | 타입 | 옵션 | 필수 |
|------|------|------|------|
| **나이대** | 단일선택 | 20대 / 30대 / 40대 / 50대+ | ✅ |
| **성별** | 단일선택 | 남성 / 여성 / 선택안함 | ⬜ |

### 1.2 여행 스타일 (페르소나)

| 스타일 | 설명 | UI 악센트 |
|--------|------|-----------|
| **럭셔리** | 돈으로 시간을 삼, 최고급 경험, 포토제닉 | Gold (#FFD700) |
| **컴포트** | 실패 없는 편안함, 안전, 검증된 맛집 | Blue (#4A90D9) |
| **백패커** | 저예산, 현지 경험, 모험 | Green (#4CAF50) |

---

## 2단계: 여행 그룹 (Group)

| 타입 | 설명 | 고려사항 |
|------|------|----------|
| **혼자** | 1인 여행 | 혼밥 가능 식당, 안전 |
| **커플** | 2인 | 로맨틱 스팟, 분위기 |
| **가족** | 3-7명 | 아이 동반 여부, 넓은 공간, 이동 편의 |
| **단체** | 7명+ | 단체 예약 가능, 대형 테이블 |

### 가족 추가 옵션

| 필드 | 타입 | 옵션 |
|------|------|------|
| **아이 동반** | 불리언 | 예 / 아니오 |
| **아이 나이** | 다중선택 | 영유아(0-3) / 어린이(4-12) / 청소년(13-18) |

## 3단계: 취향 선택 (Preferences)

> **최소 4개 이상 선택 필수**

### 3.1 음식 취향 (Food)

| 옵션 | 설명 | Taste Score 연동 |
|------|------|-----------------|
| **로컬푸드** | 현지 전통 음식 | 오리지널 언어 리뷰 가중치 ↑ |
| **파인다이닝** | 고급 레스토랑 | 미슐랭/전문가 가중치 ↑ |
| **스트릿푸드** | 길거리 음식 | 현지인 비율 가중치 ↑ |
| **카페** | 분위기 좋은 카페 | Vibe Score 가중치 ↑ |
| **채식/비건** | 채식주의 옵션 | 필터링 적용 |

### 3.2 액티비티 취향 (Activity)

| 옵션 | 설명 |
|------|------|
| **자연** | 공원, 정원, 해변, 산 |
| **쇼핑** | 백화점, 마켓, 부티크 |
| **문화/역사** | 박물관, 유적지, 건축물 |
| **나이트라이프** | 바, 클럽, 야경 |
| **사진명소** | 인스타그래머블 스팟 |
| **액티브** | 스포츠, 어드벤처 |

### 3.3 분위기 취향 (Vibe)

| 옵션 | 설명 | Vibe Score 태그 |
|------|------|----------------|
| **힐링** | 조용하고 평화로운 | relaxing, peaceful |
| **모험** | 새롭고 도전적인 | adventurous, exciting |
| **로맨틱** | 낭만적인 | romantic, intimate |
| **가족친화** | 아이와 함께 | family-friendly |
| **트렌디** | 요즘 핫한 | trendy, hip |
| **클래식** | 전통적이고 고전적인 | classic, traditional |

### 3.4 페이스 (Pace)

| 옵션 | 설명 | 일정 밀도 |
|------|------|----------|
| **여유롭게** | 하루 2-3곳 | 낮음 |
| **밸런스** | 하루 4-5곳 | 중간 |
| **빡빡하게** | 하루 6곳+ | 높음 |

## 데이터 스키마 (TypeScript)

```typescript
interface UserInput {
  // 1단계: 프로필
  profile: {
    ageGroup: '20s' | '30s' | '40s' | '50+';
    gender?: 'male' | 'female' | 'other';
    travelStyle: 'luxury' | 'comfort' | 'backpacker';
  };

  // 2단계: 그룹
  group: {
    type: 'solo' | 'couple' | 'family' | 'group';
    size?: number;
    hasChildren?: boolean;
    childrenAges?: ('infant' | 'child' | 'teen')[];
  };

  // 3단계: 취향 (최소 4개)
  preferences: {
    food: ('local' | 'fine-dining' | 'street' | 'cafe' | 'vegan')[];
    activity: ('nature' | 'shopping' | 'culture' | 'nightlife' | 'photo' | 'active')[];
    vibe: ('healing' | 'adventure' | 'romantic' | 'family' | 'trendy' | 'classic')[];
    pace: 'relaxed' | 'balanced' | 'packed';
  };

  // 4단계: 일정
  schedule: {
    destination: string;
    additionalCities?: string[];
    startDate: Date;
    endDate: Date;
    flexibility: 'exact' | 'flexible-1-2' | 'flexible';
    dailyBudget: 'budget' | 'moderate' | 'comfortable' | 'luxury';
    transportMode: 'public' | 'taxi' | 'mixed';
  };
}
```

## 알고리즘 연동

| 입력 | 영향받는 점수 | 설명 |
|------|-------------|------|
| 여행스타일 | 전체 가중치 | 럭셔리: Vibe↑, 컴포트: Taste↑ |
| 그룹 타입 | Reality Penalty | 가족: 안전↑, 접근성↑ |
| 음식 취향 | Taste Score | 로컬: 오리지널 리뷰↑ |
| 분위기 취향 | Vibe Score | 해당 태그 가중치↑ |
| 페이스 | 동선 최적화 | 일정 밀도 조절 |
| 예산 | Route Optimization | 이동수단 선택 |
