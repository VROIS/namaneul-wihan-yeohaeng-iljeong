# ✅ 미리보기 환경 체크리스트

## 🎯 목표
**미리보기가 되어야 UI/UX를 정확히 요구하고 전달할 수 있음**

---

## 📋 체크리스트

### 1단계: 환경변수 설정
- [ ] `.env` 파일 생성
- [ ] `DATABASE_URL` 설정 (Neon PostgreSQL)
- [ ] `EXPO_PUBLIC_DOMAIN=localhost:5000` 설정
- [ ] `AI_INTEGRATIONS_GEMINI_API_KEY` 설정 (선택)

### 2단계: 백엔드 서버 실행
- [ ] `npm run server:dev` 실행
- [ ] 포트 5000에서 서버 실행 확인
- [ ] DB 연결 성공 확인
- [ ] 관리자 대시보드 접근: `http://localhost:5000/admin`

### 3단계: 프론트엔드 실행
- [ ] `npx expo start` 실행
- [ ] QR 코드 또는 브라우저 접근 가능
- [ ] API 연결 확인 (에러 없음)

### 4단계: 관리자 대시보드 확인 (두뇌)
- [ ] `/admin` 접근 가능
- [ ] DB 상태 확인 (30개 테이블)
- [ ] API 서비스 상태 확인
- [ ] 데이터 수집 현황 확인

### 5단계: 프론트엔드 미리보기 (껍데기)
- [ ] 앱 화면 표시 확인
- [ ] API 데이터 로딩 확인
- [ ] UI/UX 확인 가능 상태

---

## 🔍 확인 명령어

### 서버 상태 확인
```bash
# 서버 실행 중인지 확인
curl http://localhost:5000/api/health
```

### 관리자 대시보드 확인
브라우저에서: `http://localhost:5000/admin`

### 프론트엔드 확인
- 브라우저: `http://localhost:8081`
- 또는 Expo Go 앱으로 QR 코드 스캔

---

## 🚨 문제 발생 시

### 서버가 안 열림
1. 포트 5000이 사용 중인지 확인
2. `.env` 파일 확인
3. `DATABASE_URL` 확인

### 프론트엔드가 안 열림
1. `EXPO_PUBLIC_DOMAIN` 환경변수 확인
2. 서버가 실행 중인지 확인
3. CORS 오류 확인

### DB 연결 실패
1. `DATABASE_URL` 형식 확인
2. Neon 대시보드에서 연결 문자열 복사
3. SSL 모드 확인 (`?sslmode=require`)
