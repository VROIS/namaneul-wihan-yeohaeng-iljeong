║           VibeTrip 앱 실행 방법 (비개발자용)                  ║
╚══════════════════════════════════════════════════════════════╝

📌 가장 쉬운 방법:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  start-app.bat 파일을 더블클릭하세요
   (이 파일이 보이지 않으면 파일 확장명 표시를 켜주세요)

2️⃣  검은 창이 2개 열립니다
   - 첫 번째 창: 백엔드 서버 (이 창은 그대로 두세요)
   - 두 번째 창: Expo 서버 (이 창도 그대로 두세요)

3️⃣  브라우저가 자동으로 열립니다
   - 자동으로 안 열리면 아래 주소를 직접 입력하세요

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🌐 브라우저 주소 (아래 중 하나를 시도해보세요):

   방법 1: http://localhost:8081
   방법 2: http://localhost:19006
   방법 3: http://localhost:19000

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❓ 문제 해결:

Q1. 검은 창이 안 열려요
   → Node.js가 설치되어 있는지 확인하세요
   → https://nodejs.org 에서 다운로드 가능

Q2. 브라우저가 안 열려요
   → 브라우저를 직접 열고 주소를 입력하세요
   → Chrome, Edge, Firefox 등 어떤 브라우저든 됩니다

Q3. "오류" 메시지가 나와요
   → 검은 창에 나온 오류 메시지를 확인하세요
   → 보통 "npm install"이 필요하다는 메시지입니다

Q4. 서버를 멈추고 싶어요
   → 열린 검은 창 2개를 모두 닫으세요

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 팁:
   - 처음 실행할 때는 패키지 설치로 2-5분 걸릴 수 있습니다
   - 두 번째부터는 바로 실행됩니다
   - 서버가 실행 중일 때는 검은 창을 닫지 마세요
